Password Manager

Password Manager is an Application that stores a user's passwords for all kinds of applications. In order to access all the passwords, the user needs to enter a master password. When a user wants to store a new application password, a secure password will be created for them and stored. This application was written in Python and stores the passwords in a SQLite database. All passwords are hashed using Fernet & Bcrypt and never stored as its original string.

See Repository

Reminder Pro

Reminder Pro ia an application that allows users to store their reminders on the page. Each reminder can be added or removed in real time without refreshing the browser. Users can also delete all their reminders at once if they want to get rid of all of them instead of individually removing each reminder. As the reminder approaches closer to the due date, the time for that task will be updated automatically in real time. This application was written completely with React JS.
